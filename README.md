# Edu: Seu Educador Financeiro Inteligente 🧠💰

O **Edu** é um agente de IA Generativa desenvolvido para transformar a relação das pessoas com o dinheiro. Diferente de chatbots tradicionais, o Edu atua de forma consultiva, analisando dados reais para oferecer orientações personalizadas, seguras e proativas.

Este projeto foi desenvolvido como parte do desafio **"BIA do Futuro" na DIO**.

## 🚀 Funcionalidades Principais

  - **Análise de Gastos:** Identifica padrões no histórico de transações para sugerir economias.
  - **Consultoria de Investimentos:** Recomenda produtos financeiros com base no perfil de risco do usuário.
  - **Planejamento de Metas:** Ajuda a traçar caminhos realistas para objetivos de curto e longo prazo.
  - **Segurança (Anti-alucinação):** Respostas fundamentadas estritamente na base de conhecimento fornecida.

## 📁 Estrutura do Projeto

O repositório está organizado para separar a inteligência do agente, os dados e a documentação:

  * `docs/`: Documentação detalhada da persona, tom de voz e arquitetura.
  * `data/`: Base de conhecimento com dados simulados (transações, perfis e produtos).
  * `src/`: Implementação do código e lógica do agente (prompts e integração).

## 🛠️ Tecnologias Utilizadas

  - **IA Generativa:** Configuração de System Prompts avançados para comportamento consultivo.
  - **Linguagem:** Python / SQL (para manipulação e análise da base de dados).
  - **Dados:** Estruturas em JSON e CSV para simulação de cenários reais.

## 📖 Como visualizar o projeto

1.  Acesse a pasta `docs/01-documentacao-agente.md` para entender a estratégia por trás do Edu.
2.  Confira os prompts estruturados em `docs/03-prompts.md`.
3.  Veja a lógica de processamento de dados nos scripts da pasta `src/`.
